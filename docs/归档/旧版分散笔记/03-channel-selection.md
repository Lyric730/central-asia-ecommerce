# 03 渠道选择 SOP

## 渠道分层

| 层级 | 渠道 | 阶段 | 作用 |
|---|---|---|---|
| L1 | Classifieds + 社媒 + 聊天承接 | P0 | 快速触达陌生客户，不被平台入驻卡住 |
| L2 | 本地 marketplace | P1 | 提升信任、平台内支付、自然流量 |
| L3 | Wildberries/Ozon 跨国平台 | P1/P2 | 跨国复制，但要处理仓配、退货、结算、主体 |
| L4 | 独立站 + 广告 | P1/P2 | 品牌资产、复购、批发询盘，不适合作为 P0 唯一路径 |
| L5 | 本地分销/批发 | P3 | 批量出货和线下铺货 |

## P0 渠道矩阵

| 国家 | P0 渠道 | 付款能力 | 主要问题 |
|---|---|---|---|
| Kazakhstan | OLX.kz + Instagram + Telegram/WhatsApp | 中：外部支付链接/转账 | 需要本地收款方；classifeds 不托管交易 |
| Uzbekistan | OLX Uzbekistan + Telegram + Instagram | 中：Payme/Click/本地转账 | 卖家信息披露、主体、退换规则要清楚 |
| Kyrgyzstan | Lalafo + Instagram/Facebook/Telegram | 中弱：Freedom Pay/银行转账/COD 兜底 | 现金经济明显，主动预付款转化需实测 |
| Tajikistan | Somon + Facebook/Instagram/Telegram | 弱中：Alif/本地转账 | 物流和清关不稳，只做小样 |
| Turkmenistan | Instagram/Telegram + 本地合作方 | 弱 | 不进 P0 主线 |

## P1/P2 平台矩阵

| 平台 | 国家 | 阶段 | 硬门槛 |
|---|---|---|---|
| Kaspi | Kazakhstan | P1 | Kaspi Pay；本地经营主体/账户；近期收款记录等条件 |
| Uzum | Uzbekistan | P1 | 乌兹 IP/OOO/法人/自雇；注册证、身份/任命文件；税务系统配置 |
| Olcha | Uzbekistan | P1 | 乌兹注册法人/IP/自雇 |
| Wildberries | Kazakhstan/Kyrgyzstan 等 | P1/P2 | 本地税务居民公司/IP；非居民和自然人注册受限 |
| Ozon | KZ/KG/UZ | P1/P2 | 本地 IP/法人资料、税号、地址、付款信息 |
| Flip | Kazakhstan | P1 | 合作/佣金模式，需进一步确认主体/银行资料 |
| Satu | Kazakhstan | 待确认 | 未找到可靠公开官方 seller 硬门槛 |
| Asaxiy | Uzbekistan | 合作/P2 | 未找到可靠公开第三方卖家入驻要求 |
| Gerekli | Turkmenistan | P2/P3 | 有卖家入口，但主体/税务/收款门槛未核清 |

## P0 推荐路径

### Kazakhstan

```text
OLX.kz / Instagram / TikTok / Telegram
-> WhatsApp / Telegram 私聊
-> 本地 Kaspi Pay / Halyk / 银行转账 / payment link
-> Almaty 轻仓或合作方发货
-> 记录付款、聊天、物流、售后
```

### Uzbekistan

```text
OLX Uzbekistan / Telegram / Instagram
-> WhatsApp / Telegram 私聊
-> Payme / Click / 本地银行卡 / Freedom Pay
-> Tashkent 轻仓或合作方发货
-> 记录付款、聊天、物流、售后
```

### Kyrgyzstan

```text
Lalafo / Instagram / Telegram
-> 私聊确认
-> Freedom Pay KG / 本地转账
-> Bishkek 节点发货
-> 小样本记录
```

## 免费流量和付费投放

| 类型 | 优点 | 缺点 | SOP 规则 |
|---|---|---|---|
| 免费 classifieds | 成本低、快、能听到真实问题 | 人工重，付款信任弱 | P0 必做 |
| 免费社媒内容 | 可测试场景素材和卖点 | 起量慢，归因弱 | P0 必做，配合私聊承接 |
| Telegram/WhatsApp 群 | 转化快，反馈直接 | 封号/骚扰/信任风险 | 只做合规发布，不刷屏 |
| 小额广告 | 快速测点击、询盘、国家 | 没有收款闭环会烧钱 | 只在收款/发货准备好后开 |
| Marketplace 自然流量 | 信任强，平台支付顺 | 入驻慢，竞争强 | P1 起做 |
| KOL/小博主 | 适合车内用品演示 | 难归因，虚假流量 | P0 后用已成交 SKU 测 |

## 渠道测试最小动作

每个 SKU 至少测：

- 1 条 classifieds 商品页。
- 2 条短视频素材。
- 1 条图文素材。
- 1 套私聊话术。
- 1 个明确付款方式。
- 1 个明确配送承诺。

每个渠道记录：

- 曝光/浏览。
- 咨询数。
- 加 WhatsApp/Telegram 数。
- 报价后回复数。
- 付款数。
- 客户放弃原因。
- 平均客服耗时。
