# 02 选品 SOP

## P0 选品原则

P0 不是找最大市场，而是找最容易让陌生客户付款并完成交付的 SKU。

优先级：

1. 无电池。
2. 无蓝牙/Wi-Fi/GPS/SIM/FM/RF。
3. 无充电、无电源适配、无主动电路板。
4. 小体积、轻重量、不易碎。
5. 通用型，不强依赖车型/机型。
6. 一张图或一段视频能讲清楚。
7. 售后可直接补发或退款，不需要远程技术排障。

## P0 硬淘汰项

| 淘汰项 | 原因 |
|---|---|
| 锂电池、纽扣电池、可充电电池、充电宝 | DHL/IATA 按 Dangerous Goods 管，包装、标签、申报复杂 |
| 蓝牙、Wi-Fi、GPS、SIM、无线充、遥控、RF 发射/接收设备 | 可能触发无线/射频/EAC/加密审批 |
| 充电器、电源适配器、车充、带主动电路板车载电子 | 可能触发低压电气、EMC、RoHS、售后发热风险 |
| 液体、香薰、喷雾、清洁剂、胶水、化学品 | 可能触发危险品、卫生、运输限制 |
| 食品、保健品、化妆品、医疗宣称产品 | 监管重，P0 不需要 |
| 刹车、滤芯、灯泡、雨刮、电装件等功能性汽配 | 车型匹配、安全责任、退货解释复杂 |
| 品牌仿品、疑似侵权外观/商标 | 平台风控、清关、法律风险 |

## 推荐商品池

| 商品池 | SKU | P0 判断 | 注意点 |
|---|---|---|---|
| 车载支架 | 出风口支架、吸盘支架、仪表台夹式支架、后视镜支架 | 优先 | 不带无线充电；需要实拍稳定性视频 |
| 车内收纳 | 座椅缝隙收纳、杯架扩展、挂钩、理线夹、小收纳盒 | 优先 | 控制尺寸、气味、材质、包装变形 |
| 车内内饰 | 防滑垫、防刮贴、临时停车牌、装饰条 | 优先/观察 | 避免强车型匹配 |
| 通用手机周边 | 桌面支架、车外/桌面双场景支架、清洁套装、手机挂绳 | 优先 | 低客单，可做组合包 |
| 手机壳/钢化膜 | 只做少量热机型 | 观察 | 机型管理复杂，发错货成本高 |
| 数据线 | 低功率、非夸张快充 | 观察 | 快充虚标、断线、接口兼容风险 |
| 车充/充电头 | 有证书、有品牌、有测试 | P1 后 | P0 不碰杂牌快充 |
| 蓝牙耳机/音箱/无线 CarPlay | 电子方向 | 避开 | 电池、无线、固件、兼容、售后叠加 |
| 小家电 | 水壶、吹风机、加湿器等 | 避开 | 认证、插头、电压、破损、保修 |

## 选品评分

总分 100，硬淘汰项先淘汰。

| 维度 | 权重 | 5 分标准 | 0 分/淘汰信号 |
|---|---:|---|---|
| 需求证据 | 15 | 目标平台有同类商品、评价、广告、社媒询盘 | 只有主观感觉 |
| 付款阻力 | 10 | 用户一眼懂，不用解释参数 | 要长时间教育 |
| 合规风险 | 15 | 非电、非无线、非电池、非安全件 | 电池、无线、电源、医疗、食品 |
| 物流友好 | 12 | <500g、小体积、不易碎 | 大件、玻璃、液体、易压坏 |
| 售后复杂度 | 15 | 坏了可补发/退款 | 涉及发热、失灵、兼容排障 |
| 兼容复杂度 | 10 | 通用型 | 强机型/车型/协议依赖 |
| 毛利空间 | 10 | 能覆盖广告、补发、退货 | 低价红海无组合空间 |
| 供应链稳定 | 8 | 可稳定补货，有一致包装和质检 | 批次差异大 |
| 内容展示成本 | 5 | 一张图/一段视频能讲清 | 需要复杂说明 |

进入 P0 的最低线：

```text
无硬淘汰项
总分 >= 75
合规风险 >= 4/5
售后复杂度 >= 4/5
兼容复杂度 >= 4/5
```

## 选品执行步骤

1. 列 30 个候选 SKU。
2. 用硬淘汰项先删。
3. 剩余 SKU 用 [templates/sku_scorecard.csv](templates/sku_scorecard.csv) 打分。
4. 选 5-8 个 P0 SKU。
5. 每个 SKU 准备：
   - 实拍图 5 张。
   - 15-30 秒使用视频 1 条。
   - 俄语商品标题和 3 个卖点。
   - 尺寸、材质、产地、重量、包装尺寸。
   - 售后规则：坏件补发/退款阈值。
6. 先不要超过 8 个 SKU。P0 的重点是链路，不是铺货。

## 套装策略

P0 可以做低复杂度组合：

| 套装 | 组成 | 目的 |
|---|---|---|
| 车内整理套装 | 理线夹 + 挂钩 + 防滑垫 | 提高客单，仍保持低合规 |
| 手机支架套装 | 车载支架 + 桌面支架 | 场景互补，容易拍视频 |
| 车主基础套装 | 临时停车牌 + 防滑垫 + 收纳夹 | 低客诉，适合社媒展示 |

不要做：

- 车载支架 + 无线充。
- 收纳件 + 香薰液。
- 支架 + 车充。
- 多机型手机壳大礼包。
