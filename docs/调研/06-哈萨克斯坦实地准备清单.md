# 哈萨克斯坦实地准备清单：合伙人小白版

复核日期：2026-06-29

给谁看：马上去 Kazakhstan / Almaty 的合伙人

目标：以开 Kaspi 网店为最终目标，把在当地必须办好的东西一次列清楚。

## 0. 先把话说人话

我们不是一到哈萨克斯坦就能直接开 Kaspi 网店。

Kaspi 官方规则里，想开 Kaspi 店，前面先要做一件事：

```text
先开 Kaspi Pay
再用 Kaspi Pay 做至少 20 天真实收款
然后才有资格申请 Kaspi 网店
```

这 20 天不能靠自己买自己的货，也不能让员工刷单，退货订单也不算。[R1][R2]

所以这次合伙人去哈萨克斯坦，真正要完成的是 5 件事：

| 顺序 | 要办的事 | 为什么 |
|---:|---|---|
| 1 | 找到可用的本地主体 | 没有当地公司/个体/靠谱合作方，就很难开 Kaspi Pay |
| 2 | 办好本地手机号、IIN、电子签名 | 本地账号、银行、Kaspi、政府网站都可能要用 |
| 3 | 开通或确认 Kaspi Pay 路径 | 这是后面申请 Kaspi 店的前置条件 |
| 4 | 准备本地存货、配送、退货地址 | 开店后要能发货、退货，不能只说“客户自提” |
| 5 | 准备商品资料 | 商品要能上架、能证明来源、能处理合规问题 |

## 1. 这次去当地的最低成果

如果合伙人回来时，只能带回来一张表，就带这张。

| 必须拿回来 | 具体要拿什么 | 用途 | 合格证据 |
|---|---|---|---|
| 1 个本地主体方案 | 本地个体、当地公司，或可信合作方 | 开 Kaspi Pay / 收款 / 报税 | 主体名称、负责人、BIN/IIN、联系方式 |
| 1 条 Kaspi Pay 路径 | 已开通、正在申请，或明确缺什么资料 | 后面做 20 天真实收款 | Kaspi Pay 截图、申请记录、经理回复 |
| 20 天真实收款计划 | 用什么商品、什么渠道、每天怎么产生真实订单 | 满足 Kaspi 店申请前置 | 表格：日期、订单、客户、金额、证据 |
| 2 个以上 +7 本地手机号 | 1 个绑 Kaspi/银行，1 个做客服 | 注册、验证码、客户联系 | SIM 截图、号码归属、能收验证码 |
| IIN 方案 | 外籍负责人是否已申请 IIN，没申请就明确怎么申请 | 政府、银行、部分本地服务 | IIN 申请记录或办理清单 |
| 本地存货地址 | Almaty 能放 50-100 件小商品的地方 | 本地现货、发货、退货 | 地址、照片、负责人、收费 |
| 本地配送方案 | 至少 2 条：自己送/跑腿/Kaspi Delivery/Express | 发货给客户 | 价格、时效、测试单 |
| 退货地址 | 客户退货、坏件处理、补发都发到哪里 | 售后和评分 | 地址、负责人、退货流程 |
| 5 个首批 SKU 资料 | 图、标题、价格、库存、采购证明 | 先做低风险商品 | SKU 表、照片、发票/合同 |
| 订单台账 | 每一笔订单、付款、发货、售后都记录 | 后续复盘和对账 | Google Sheet / Airtable / Notion 链接 |

## 2. 出发前要带什么

### 2.1 人的材料

| 材料 | 要不要带 | 用途 |
|---|---|---|
| 护照原件 | 必带 | 办 IIN、手机号、银行、本地认证 |
| 护照扫描件 | 必带 | 各种线上申请和备份 |
| 护照俄语/哈语公证翻译 | 强烈建议 | IIN、公司注册、银行补件可能要 |
| 入境/合法停留证明 | 必带 | 银行、Kaspi Gold、部分 KYC 会问 |
| 中国手机号 | 必带 | 国内账号、微信、银行、邮箱验证 |
| 能收邮件的工作邮箱 | 必带 | Kaspi、物流、合作方、合同往来 |
| 授权委托书 | 如果合伙人不是老板本人，就带 | 代表团队谈主体、银行、仓库、服务商 |

IIN 是哈萨克斯坦给个人的识别号。外国人可以申请，官方页面显示服务免费，通常 1 个工作日，材料包括申请和有效护照；如果证件信息不符合要求，可能需要公证翻译。[R9]

### 2.2 公司材料

如果只找本地合作方代跑，下面材料可以先带电子版。

如果要注册自己的 Kazakhstan 公司，下面材料要准备正式版。

| 材料 | 用途 |
|---|---|
| 中国公司营业执照 |
| 公司章程 |
| 股东和实际控制人名单 |
| 法定代表人/负责人身份证明 |
| 授权合伙人在 Kazakhstan 办事的文件 |
| 公司介绍：卖什么、从哪里进货、怎么配送、怎么售后 |
| 以上材料的英文/俄文版本，必要时做公证翻译 |

外国公司或外国人参与当地公司注册时，eGov 对外资注册资料有要求，包括商业登记摘录或同等文件、外国创始人护照复印件、哈语/俄语公证翻译等。[R8]

### 2.3 商品材料

首批只准备低风险商品：

```text
非通电车载支架
车内收纳
理线夹
防滑垫
普通手机支架
普通手机周边小件
```

暂时不要把下面这些作为首批：

```text
充电宝
蓝牙耳机
车充
无线 CarPlay
智能手表/手环
小家电
液体香薰
带强安全责任的汽配
```

每个 SKU 要准备：

| 材料 | 用途 |
|---|---|
| 采购合同或供应商合同 | 证明货从哪里来 |
| 发票/送货单 | 证明采购和库存 |
| 付款证明 | 证明真实采购 |
| 实拍图片 | 上架、聊天、广告、售后都要用 |
| 俄语标题和卖点 | 给本地客户和 Kaspi/OLX 使用 |
| 重量、尺寸、成本、建议售价 | 算运费、仓储、利润 |
| HS code 初判 | 后面进口、税费、认证要用 |
| 条码/GTIN/NTIN 准备 | Kazakhstan 商品目录和 Kaspi 上架会越来越需要 |

## 3. 到当地先办哪几件事

### 第 1 件：办手机号和设备注册

最低准备：

| 号码 | 用途 |
|---|---|
| +7 号码 A | 绑定 Kaspi Pay、银行、政府网站、2FA |
| +7 号码 B | WhatsApp Business / Telegram 客服 |
| +7 号码 C | 备用恢复号，完整体建议准备 |

注意：

```text
不要用一次性接码。
不要用老板私人号做客服。
不要所有平台共用一个号码。
不要让本地合作方独占绑定手机号。
```

Kcell 页面说明，外国人没有 IIN 时可以用护照信息注册设备；有 IIN 后可用 USSD/SMS 等方式注册设备。[R7]

### 第 2 件：确认主体怎么解决

只有 3 种路线：

| 路线 | 是什么 | 适合什么时候 | 风险 |
|---|---|---|---|
| 本地合作方 | 用当地已有个体/公司帮我们开 Kaspi Pay、收款、发货 | MVP 最快 | 钱、库存、账号可能被对方卡住 |
| 自己注册 Kazakhstan 公司 | 我们自己控制当地公司，再开 Kaspi Pay | 完整体最佳 | 慢，需要会计、税务、银行 |
| 中国/境外公司直接尝试 | 问 Kaspi 能不能按非居民接入 | 只能并行问 | Kaspi 逐案决定，不稳定 [R5][R6] |

小白执行建议：

```text
MVP：可以先用本地合作方，但必须签清楚钱、货、账号、客户归属。
完整体：尽量走自有 Kazakhstan 公司。
不要把项目押在“境外公司直接开 Kaspi”这一条上。
```

### 第 3 件：开 Kaspi Pay，开始准备 20 天真实收款

Kaspi Pay 是本地收款工具。

Kaspi 店铺申请前，要先满足：

```text
已经是 Kaspi Pay 伙伴
最近 2 个月内至少 20 天通过 Kaspi Pay 收款
销售的是商品，不是服务
```

不计入 20 天的情况：

```text
自己买自己的货
员工买
客户退货
```

这 20 天怎么做才靠谱：

| 动作 | 要求 |
|---|---|
| 用真实商品 | 先用 5 个低风险 SKU |
| 用真实客户 | OLX、Instagram、熟人转介绍都可以，但要是真客户 |
| 每天留证据 | 聊天记录、付款截图、出库照片、客户确认 |
| 每天记台账 | 日期、客户、商品、金额、付款方式、发货方式 |
| 不刷单 | 刷单会把后面店铺和财务都带歪 |

### 第 4 件：准备本地仓配和退货

2026-06-15 之后，新接入 Kaspi 店铺的卖家，自提功能不可用。也就是说，不能只准备“客户来拿货”。要准备配送。[R3][R4]

最低要准备：

| 项 | 要求 |
|---|---|
| 存货点 | Almaty 可放 50-100 件小商品 |
| 出库人 | 谁负责按订单找货、拍照、打包 |
| 配送方式 1 | 本地跑腿/自己送 |
| 配送方式 2 | Kaspi Delivery 或 Express delivery |
| 退货地址 | 客户退货、坏件、补发都回到哪里 |
| 包装材料 | 快递袋、气泡袋、标签纸、胶带 |
| 打印设备 | 普通打印机或热敏打印机，用来打面单 |

Kaspi Express delivery 要求卖家收到提醒后 1 小时内完成包装并贴单，由 Kaspi 指派快递上门取货。[R20]

### 第 5 件：准备商品上架和合规

首批只做低风险商品，是为了少踩坑。

但完整体必须补齐：

| 项 | 为什么要准备 |
|---|---|
| 条码 / GTIN / NTIN | Kazakhstan 商品目录和 Kaspi 上架会用到 [R16][R17] |
| 采购合同 / 发票 / 付款证明 | Kaspi 限制品类和进口举证可能要 [R18] |
| EAC / Declaration | 带电、无线、电池、小家电后续可能要 |
| HS code | 进口、税费、监管都要看 |
| 商业进口方案 | 商业货不要套个人自用包裹规则 [R19] |

```text
  08-Go-NoGo决策/
```

## 5. 现场要问的 20 个问题

### 问 Kaspi / Kaspi Pay

1. 我们这个主体能不能开 Kaspi Pay？
2. 如果是中国公司/境外公司，能不能接入？要什么文件？
3. 20 天收款在哪里看？系统里有没有进度？
4. 哪些付款方式算 20 天？
5. 哪些订单不算？
6. 到 20 天后，谁能申请 Kaspi 店？
7. 被拒后多久能重新申请？
8. 新卖家现在能不能自提？
9. 我们这种车载支架/收纳小件有没有限制？
10. 上架要不要条码、NKT、GTIN、NTIN？

### 问本地合作方

1. 你有没有 Kaspi Pay？
2. 最近有没有真实收款记录？
3. 能不能给我们看收款截图和主体信息？
4. 能不能帮我们做 20 天真实收款？
5. 钱多久结给我们？
6. 你能不能存货、出库、发货？
7. 退货谁收？
8. 丢件、坏件谁赔？
9. 合作结束后，库存、客户、账号怎么交接？
10. 你是否接受每天对账？

## 6. Go / No-Go 判断

### 可以继续推进 Kaspi

满足下面这些，才继续推进：

```text
有可控本地主体或高可信合作方
Kaspi Pay 能开，或缺什么资料已经明确
有 2 个以上 +7 本地手机号
有本地存货地址
有本地配送方式
有退货地址
有 5 个低风险 SKU 资料
有订单和收款台账
有 20 天真实收款计划
```

### 暂停 Kaspi，先跑普通本地闭环

出现下面任一情况，先别硬开 Kaspi：

```text
没有本地主体，也没有靠谱合作方
Kaspi Pay 开不了，原因不清楚
合作方只愿意收钱，不愿意对账
没有仓库/发货/退货地址
20 天计划靠刷单
首批 SKU 涉及带电、无线、电池，但没有认证资料
```

### 直接 No-Go

出现下面任一情况，直接停：

```text
合作方要求账号、库存、资金都由他控制
合作方不给主体证明
合作方不给 Kaspi Pay 权限截图
合作方不接受每日对账
合作方要求低报、错报、灰色清关
有人承诺“包开 Kaspi 店”，但解释不清 20 天 Kaspi Pay 规则
```

## 7. 名词解释

| 名词 | 是什么 |
|---|---|
| Kaspi | 哈萨克斯坦最核心的支付、电商、金融平台之一 |
| Kaspi Pay | 给商户收款用的 Kaspi 工具 |
| Kaspi 店 | 在 Kaspi.kz 上卖货的店铺 |
| IIN | 哈萨克斯坦个人识别号，外国人也可以申请 |
| BIN | 哈萨克斯坦企业识别号 |
| IP | 哈萨克斯坦个体经营者 |
| TOO / LLP | 哈萨克斯坦有限责任公司 |
| EDS | 电子签名，用来在政府网站和部分平台线上签文件 |
| NKT | Kazakhstan 国家商品目录 |
| GTIN / NTIN | 商品条码/商品识别码 |
| EAC | 欧亚经济联盟相关合规标志 |
| HS code | 海关商品编码 |
