# 05 支付、履约、清关、售后 SOP

## 默认履约原则

P0 优先选择：

```text
中国小批量备货 -> 目标国本地节点/轻仓 -> 本地派送/自提
```

不优先选择：

```text
中国逐单直发 -> 陌生客户
```

原因：

- 时效慢会降低陌生客户预付款意愿。
- 退换货难。
- 海关/税费/签收责任不清。
- 客服解释成本高。

## 国家支付/履约矩阵

| 国家 | 支付路径 | 履约路径 | P0 策略 |
|---|---|---|---|
| Kazakhstan | Kaspi Pay、Halyk EPAY、本地银行转账；需要本地主体/账户 | KZ 小包个人限额 €200/31kg；CDEK/DHL/本地派送；DHL DDP 对无 KZ BIN 外企有限制 | 强市场，但先确认本地收款方 |
| Uzbekistan | Payme、Click、Freedom Pay、本地银行卡/钱包 | Courier $200/月/收件人；postal $100；Tashkent 节点优先 | 如果收款/轻仓先就绪，可优先跑 P0 |
| Kyrgyzstan | Freedom Pay KG、本地钱包/银行转账 | EAEU 口径 €200；CDEK 可查 | 第二梯队，小样本 |
| Tajikistan | Alif acquiring、本地转账 | DHL/CDEK 可查，但进口复杂，个人小包限额未找到可靠公开来源 | 只做小样验证 |
| Turkmenistan | 未找到可靠公开在线收款路径 | 清关/许可不透明，个人电商小包限额未找到可靠公开来源 | P0 排除 |

## 清关/限额

| 国家 | 当前可用规则 | SOP 处理 |
|---|---|---|
| Kazakhstan | 个人小包 €200 / 31kg；超出按 15% 超额或至少 €2/kg | 单票控制在 €200 以下；不包税；真实申报 |
| Kyrgyzstan | EAEU 口径 €200；2026-07-01 附近 EAEU e-commerce 新制需复核 | 单票 <€200；执行前复核新规 |
| Uzbekistan | Courier $200/月/收件人；postal $100 | 客单和申报价控制；记录收件人手机号 |
| Tajikistan | 未找到可靠公开个人小包限额 | 按可能征税/查验设计；先让清关方报价 |
| Turkmenistan | 未找到可靠公开电商小包限额 | 不进 P0；必须 broker 预确认 |

## SKU 合规处理

| SKU | P0 处理 | 原因 |
|---|---|---|
| 机械车载支架 | 可做 | 低合规、低售后 |
| 车内收纳/挂钩/理线夹 | 可做 | 小件、低风险 |
| 手机桌面/车载支架 | 可做 | 通用场景 |
| 手机壳/膜 | 观察 | 型号管理复杂 |
| 数据线 | 观察 | 快充虚标和断线风险 |
| 车充/充电头 | 不做 P0 | 电气安全、EMC、发热售后 |
| 蓝牙/无线设备 | 不做 P0 | 无线/射频/电池/固件 |
| 充电宝/带电池产品 | 不做 P0 | 锂电运输和危险品 |
| 小家电 | 不做 P0 | 电气认证、插头、电压、破损、保修 |
| 液体/香薰/清洁剂 | 不做 P0 | 危险品/化学品/运输限制 |

## 发货前检查

每个 SKU 发货前确认：

- SKU 名称。
- HS code 初判。
- 申报价。
- 单件重量。
- 包装尺寸。
- 材质。
- 是否含电池。
- 是否含无线/RF。
- 是否含电路板/充电功能。
- 是否液体/化学品。
- 目标国限额。
- 目标国标签语言要求是否需要额外确认。

## 售后策略

P0 不做复杂退货维修，采用简单规则：

| 情况 | 处理 |
|---|---|
| 发错货 | 补发或退款 |
| 到货破损 | 客户提供照片/视频后补发或退款 |
| 客户说不喜欢 | P0 可拒绝无理由退货，除非渠道/平台要求 |
| 支架不稳/不适配 | 先看安装视频；确属产品问题补发/退款 |
| 超时未到 | 按物流状态判断；超过承诺上限给补偿/退款 |

推荐 P0 规则：

```text
低客单 SKU：优先补发，不做跨境退回。
高争议 SKU：不要进入 P0。
补发/退款成本计入真实获客成本。
```

## 合作方最低要求

本地合作方必须能提供：

- 收款方式。
- 对账截图。
- 退货地址。
- 发货能力。
- 物流报价。
- 身份/主体资料。
- 每周对账。
- 不挪用货款的基本约束。
