# Central Asia Ecommerce MVP SOP

中亚五国小宗商品出海电商 MVP SOP。

数据冻结日期：2026-06-15  
当前交付等级：🟡 可用  
默认策略：先跑通 P0/P1，不追求一开始五国铺开或稳定利润。

## 核心结论

| 项 | 结论 |
|---|---|
| P0 首站默认 | Kazakhstan |
| P0 第二市场 | Uzbekistan |
| 可低成本跟进 | Kyrgyzstan |
| 只做小样验证 | Tajikistan |
| P0 排除 | Turkmenistan |
| P0 商品 | 非通电车内用品、机械车载支架、通用手机周边 |
| P0 渠道 | Classifieds + 社媒/聊天 + 本地收款 |
| P1 渠道 | Kaspi / Uzum / Olcha / Ozon / Wildberries / Flip 等 marketplace |

## 执行前复核

如果距离数据冻结日期超过 30 天，执行前必须复核：

- 平台入驻规则。
- 支付产品可用性。
- 清关/小包限额。
- 广告投放国家列表。
- 目标 SKU 的 HS code、认证和禁限售。
