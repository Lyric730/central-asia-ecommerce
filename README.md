## 当前默认结论

| 项 | 默认选择 |
|---|---|
| MVP 首站 | Kazakhstan |
| 备选首站 | Uzbekistan |
| 先做城市 | Almaty |
| 先做商品 | 非通电车载支架、车内收纳、理线夹、防滑垫、通用手机支架 |
| 先做渠道 | TikTok + Instagram + OLX.kz + WhatsApp/Telegram + 落地页/CRM |
| 先做收款 | 本地合作方 Kaspi Pay/payment link/银行转账 |
| 先做履约 | 中国小批量备货到 Almaty 节点，再本地派送/自提 |
| 不做 | 带电池、蓝牙/Wi-Fi/RF、车充、充电宝、小家电、液体、香薰、功能性汽配 |

## 执行前复核

如果距离数据冻结日期超过 30 天，执行前必须复核：

- 平台入驻规则。
- 支付产品可用性。
- 清关/小包限额。
- 广告投放国家列表。
- 目标 SKU 的 HS code、认证和禁限售。
